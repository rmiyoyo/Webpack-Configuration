# Webpack Configuration
Includes configuration code for:
<ul>
    <li>Webpack</li>
    <li>Linters
        <ol>
            <li>Lighthouse</li>
            <li>Webhint</li>
            <li>Stylelint</li>
            <li>ESlint</li>
        </ol>
    </li>
</ul>

## Prerequisites

In order to run this project you need:

- A browser
- Node
- npm

## Setup

Clone this repository to your desired folder:


```sh
cd my-folder
git clone git@github.com:rmiyoyo/Webpack-Configuration.git
```

## Installation
```bash
npm install
```

## Usage

To run the project:

```bash
npm start
```

## Linters
```sh
npx hint .
```
```sh
npx stylelint "**/*.{css,scss}"
```
```bash
npx eslint .
```
